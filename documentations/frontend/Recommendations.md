# Recommendations Page Component

## Overview
The `Recommendations.tsx` file displays personalized gift recommendations for the user. It uses AI-powered algorithms to suggest products based on user preferences.

## Key Features
- Displays a list of recommended products.
- Highlights the top recommendation with a badge.
- Allows users to select a product and proceed to the delivery page.

## Structure
- **Recommendation List**: Shows a list of recommended products with details such as price, store, and match score.
- **Baseline Comparison**: Compares AI recommendations with baseline products.

## Notes
- The recommendations use default data if backend data is unavailable.
- The baseline comparison feature provides insights into the effectiveness of AI recommendations.

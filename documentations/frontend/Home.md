# Home Page Component

## Overview
The `Home.tsx` file implements the homepage of the ZIPPO frontend. It showcases top picks, categories, and a search bar for users to find gifts based on occasions.

## Key Features
- Displays personalized top picks for the user.
- Provides a search bar for catalog search.
- Includes occasion-based filtering options.

## Structure
- **Top Picks**: Displays a list of recommended products.
- **Occasion Filters**: Allows users to filter products by occasion.
- **Search Bar**: Enables users to search for specific products.

## Notes
- The top picks currently use fallback data. This should be replaced with dynamic data from the backend.
- Occasion filters and categories are hardcoded and should be fetched dynamically in the future.
